# RNSUV - JokeHub Project

University project for RNSUV course demonstrating Angular framework.

## Project Structure

| Folder | Description | Documentation |
|--------|-------------|---------------|
| `joke-hub/` | frontend (Angular) | [README](joke-hub/README.md) |
| `joke-hub-server/` | backend     | [README](joke-hub-server/README.md) |

## Quick Start

**1. Start backend:**
```bash
cd joke-hub-server
npm install
npm start
```

**2. Start frontend (new terminal):**
```bash
cd joke-hub
npm install
ng serve
```

**3. Open:** http://localhost:4200
